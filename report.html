Coursework Project Report

1. Overview

This website is designed as a modern portfolio-style coursework project with three main pages: Home, About, and Multiplayer Puzzle Game. The visual style follows Apple-inspired minimalism with pastel pink and white as the core palette.

2. Technical Stack

The project uses only required technologies: HTML5, CSS3, Vanilla JavaScript, Node.js, Express, and Socket.io. No extra dependencies were added.

3. UI/UX Design Strategy

The UI focuses on clean hierarchy, whitespace, and elegant typography. Advanced but lightweight effects are implemented, including particle background, cursor glow, 3D hover cards, smooth transitions, and scroll reveal animation.

4. Multiplayer Game Logic

  • 4x4 board and 16 unique blocks (4 shapes x 4 colors)
  • Turn-based rotation with dynamic player list
  • Random block assignment to active player only
  • Line match detection (horizontal/vertical/diagonal)
  • Block removal + score increments
  • Full board jackpot: +16 points and full clear
  • Timeout/disconnect handling for robust session state

5. Reflection

The main challenge was synchronizing real-time state updates while preserving a premium visual experience. Separating global visual scripts and game-specific scripts improved maintainability and clarity.

6. Run Instructions

Install and run with npm install and npm start, then open http://localhost:8080.
